# Contributing

Thank you for your interest in contributing to this project! We welcome pull requests, issues, and feedback from anyone passionate about the developer tools and AI agents ecosystem.

## How to Contribute

1. **Fork** the repository and create your branch from `main`.
2. Make your changes — whether it's fixing a typo, improving the design, or adding new content.
3. **Test locally** by opening `index.html` in a browser or using a local static server.
4. Submit a **pull request** with a clear description of what you changed and why.

## What We're Looking For

We're especially interested in contributions from people who understand:

- **Developer workflows** — How developers discover, evaluate, and adopt tools.
- **AI coding agents** — The current landscape (GitHub Copilot, Cursor, Codeium, Devin, SWE-Agent, etc.), their strengths, and their limitations.
- **Market dynamics** — How the AI agents market is evolving, where the opportunities are, and what developers actually need.

Insightful issues and PRs that demonstrate deep understanding of these areas are highly valued.

## Guidelines

- Keep changes focused and minimal.
- Use plain HTML and CSS — no frameworks or build tools unless there's a strong reason.
- Write clear commit messages.
- Be respectful and constructive in discussions.

## Reporting Issues

If you spot a bug, have a suggestion, or want to discuss an idea, please [open an issue](https://github.com/athamdi/athamdi.github.io/issues). Include as much context as possible so we can understand and act on your feedback.
