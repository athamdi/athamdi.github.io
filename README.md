# athamdi.github.io

Personal website for [atharvarakhe.me](https://atharvarakhe.me) — a lightweight, static site hosted on GitHub Pages.

## Overview

This repository powers a personal landing page focused on the intersection of **developer tools** and **AI agents**. It's built with plain HTML and CSS (no frameworks, no build step) for simplicity and fast load times.

## Local Development

To preview the site locally, serve the repository root with any static file server:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js (npx)
npx serve .
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.
